Hanche 3D
========================

# Langages utilisés

* HTML5, SCSS
* Typescript
* ThreeJS

# Framework

* Angular 10

# Dépendances

Le projet inclus les bibliothèques tierces suivant :

* RxJS
* ThreeJS
* ThreeCSG
* Firebase
* Angular Material
* doodle3d/threejs-export-stl
* Three-csg-ts
* Three-js-csg-es6
* Threejs-export-stl
* Camera-controls
* Bootstrap
* Font-awesome


# Installation

## Pour commencer cloner le dépôt

## Installation des dépendances du projet

 `npm install`
    
## Lancer l'application

`ng serve`

## Deploiement de l'application sur Firebase

`ng deploy`

## Lancement des tests unitaire 

`ng test`
